Website Development and Branding Information

Pegasus Construction Services, LLC

This site is a summary of the visual identity and styling information regarding the web development and branding of Pegasus Construction Services, LLC.

Branding

The logo of the web page is a stylized with the company name with four wings, construcntion craine and house.

It represents the name and the concept of the web page, which is Pegasus, a mythical winged horse. The logo has a blue color (#0078d4) that matches the primary color of the web page.

Primary Colors:

#4a8cff: This vibrant blue shade serves as the primary color for Pegasus Construction Services. It represents professionalism, reliability, and trustworthiness. It can be used for important elements like headings, buttons, or accents.

#0078d4: This deep blue color complements the primary color and can be used for various design elements. It exudes a sense of stability, sophistication, and authority. It can be used for backgrounds, call-to-action buttons, or navigation elements.

Neutral Colors:

#364d59: This dark blue-gray hue works well as a neutral color to provide contrast and balance in the design. It can be used for text, secondary elements, or backgrounds. It adds depth and richness to the overall visual presentation.

#52565e: Can be utilized as a secondary or supporting color. It works well for elements that require a slightly darker tone than the primary colors, such as subheadings, dividers, or secondary buttons. This color adds depth and contrast to the overall design, helping to create visual interest and hierarchy.

#ffffff: This pure white color serves as the primary background color, providing a clean and fresh look. It creates a sense of spaciousness and ensures readability. It can be used as the background color for the main content area or sections.

#000000: This solid black color can be used for text and other important elements that require high contrast. It adds emphasis and readability to the content and helps in creating visual hierarchy.

The typography of the web page is based on Segoe UI, which is a sans-serif font designed by Microsoft. It is used for the headings, the body text, and the labels. The font size varies from 14px to 36px depending on the hierarchy and importance of the text. The font weight varies from 300 to 600 depending on the emphasis and style of the text. The font color varies from black to white depending on the contrast and readability of the text.

The typography of the web page should be consistent, clear, and modern. It should create a professional and friendly impression for the users.

Site Layout

The layout of the web page is based on a responsive design that adapts to different screen sizes and devices. It uses media queries, flexbox, and grid layout to achieve this. The layout consists of four main sections:

Header:

This section contains the logo, the navigation bar, and a banner image with a slogan.

Main:

This section contains the main content of the web page, which is a contact form with some information and icons.

Sidebar:

This section contains some additional information and icons that appear when clicked on a button. The layout of the web page is simple, functional, and user-friendly. It creates a smooth and intuitive user experience.

Footer:

This section contains some links and social media icons.

Animation:

The animation of the web page is based on some transitions and transformations that create a dynamic and engaging user experience. The animation consists of four main effects:

Logo rotation:

The logo rotates when hovered over, creating a playful effect.

Button color change:

The buttons change color when clicked, creating a feedback effect.

Image slide in:

The images slide in when scrolled, creating a motion effect.

Sidebar slide out (Mobile):

The sidebar slides out when clicked on a button, creating an expansion effect.

Development

The Pegasus website uses Bootstrap version 5.3.0, which can be installed via package manager, CDN or downloaded source files. The website includes the following Bootstrap files:

bootstrap.min.css:

The compiled and minified CSS file that contains the core styles and components of Bootstrap.

bootstrap.bundle.min.js:

The compiled and minified JS file that includes every Bootstrap JavaScript plugin and dependency. The website also uses some custom CSS styles that override or complement the Bootstrap styles.

The Pegasus website uses jQuery version 3.6.0, which can be included via CDN or downloaded source file. The website includes the following jQuery file:

jquery-3.6.0.min.js:

The compressed and minified production version of jQuery3. The website also uses some custom JavaScript scripts that utilize the jQuery methods and features
